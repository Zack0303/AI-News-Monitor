# GROWTH ROADMAP (CN AUDIENCE)

## P0 (本周)
1. 稳定内容质量
- 强制字段：`why_it_matters`、`next_action`
- 噪音降权：营销稿、重复稿、低信息密度内容

2. 偏好反馈闭环上线
- 手动投喂 + 页面点赞/点踩 + 画像更新

3. 发布质量门禁
- CI + 发布前检查 + 发布后检查

## P1 (2-4 周)
1. SEO 增长基础
- 结构化数据（Article）
- sitemap
- 卡片级可索引详情页

2. 订阅与留存
- 邮件订阅入口
- 每周精选自动生成
- 首页显著 CTA

3. 分发矩阵
- 自动导出公众号/知乎/即刻版本摘要

## P2 (1-2 月)
1. 角色化首页
- 工程师 / 产品 / 投资三种视图

2. 社区互动
- “本周最值得跟进”投票
- 用户反馈榜单

3. 数据看板
- 点击率、回访率、订阅转化率
- 标签表现与来源表现

## 成功指标
1. 7 日回访率
2. 单次会话点击原文数
3. 订阅转化率
